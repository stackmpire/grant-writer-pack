How it works (no account needed)

Grant Writer Pack is a local Claude Code plugin — there's no dashboard to log into. Your license key is your access credential. It lives in one config file on your machine. Here's the 3-step setup:

  1. 1
    Clone the repo & copy your profile

    Open Terminal and run:

    # Clone (or pull if you already have it)
git clone https://github.com/stackmpire/grant-writer-pack.git
cd grant-writer-pack
# Copy the example profile
cp config/profile.example.json config/profile.json
  2. 2
    Paste your license key into profile.json

    Open config/profile.json in any text editor and add your key (from the email) to the "license_key" field. Also fill in your keywords and preferred agencies — this is what the ranker uses to score opportunities for your niche.

  3. 3
    Open in Claude Code and scan 
    # Open the folder in Claude Code
claude .
# Then type in the Claude Code chat:
/grants-scan

    Your first ranked shortlist arrives in about 30 seconds.

Three things to do in your first 10 minutes

🎯
Tune your profile

Add your keywords, preferred agencies and ALN numbers to profile.json. Re-run /grants-scan — results sharpen immediately.

📂
Add past proposals

Drop 2–3 past proposals as .txt or .md into the boilerplate/ folder. /grants-boilerplate needs something to pull from.

📄
Try /grants-draft

Paste the URL of any open RFP you're looking at. Get a full Markdown skeleton with sections, word limits, and the evaluation rubric in under a minute.

⏰
Set up the daily scan

Follow the LaunchAgent setup in the README to get your ranked shortlist waiting in data/ranked.json every morning at 7 AM — automatically.
